# openvpn-mgmt

Cargo workspace for the OpenVPN management interface toolkit.

| Crate | Description |
| --- | --- |
| [`openvpn-mgmt-codec`](openvpn-mgmt-codec/) | Tokio codec, typed commands/messages, parsers, and high-level `ManagementClient`. Published to crates.io. |
| [`openvpn-mgmt-cli`](openvpn-mgmt-cli/) | Interactive CLI for talking to an OpenVPN management socket. |
| [`openvpn-mgmt-ui`](openvpn-mgmt-ui/) | Terminal UI for monitoring OpenVPN connections. |

See the [codec README](openvpn-mgmt-codec/README.md) for API documentation and usage examples.
